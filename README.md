## How to Use :gear:

1. Clone the repository:

```bash
git clone https://github.com/KaranChandekar/nodejs.git
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm start
```
or
```bash
npm run dev
```

4. Open your browser and go to:

```bash
http://localhost:5000
```
